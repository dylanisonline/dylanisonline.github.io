Role Groups


Role Groups

Role Description
Kojima-GG-Role-SysAdmin-DA Domain access role for those with SysAdmin title or above
Kojima-GG-Role-SysAdmin-WS Workstation access role for those with SysAdmin title or above
Kojima-GG-Role-SysAdmin-SA Server access role for those with SysAdmin title or above


Creating Role Groups via PowerShell:


Import-Module ActiveDirectory

$GroupPath = "OU=Admin Roles,OU=Kojima Roles,OU=Kojima,DC=kojima,DC=test"

try {
    New-ADGroup `
        -Name "Kojima-GG-Role-SysAdmin-DA" `
        -SamAccountName "Kojima-GG-Role-SysAdmin-DA" `
        -GroupCategory Security `
        -GroupScope Global `
        -Path $GroupPath `
        -Description "Role Group for System Administrator Domain Access" `
        -ErrorAction Stop

    Write-Host "Global Role groups created successfully." -ForegroundColor Green
}
catch {
    Write-Host "Group creation failed." -ForegroundColor Red
    Write-Host $_.Exception.Message -ForegroundColor Yellow
}
