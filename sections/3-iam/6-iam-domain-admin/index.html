Domain Administrator Account


Due to Hal's title of 'Systems Engineer' he will require a high level of access and control of domain resources.


Account: hemmerich-da

Permissions:

  • Manage User accounts (create, delete, modify, membership, reset passwords)
  • Group Management (Create, delete, modify)
  • OU management (Create, delete, modify)
  • GPO management (Create, delete, modify)
  • Computer Management (Create, delete, modify)


Applying task roles to Domain Administrator Role


Import-Module ActiveDirectory

try {
    Add-ADGroupMember `
        -Identity "Kojima-GG-User-Management" `
        -Members "Kojima-GG-Role-SysAdmin-DA"

    Add-ADGroupMember `
        -Identity "Kojima-GG-Computer-Management" `
        -Members "Kojima-GG-Role-SysAdmin-DA"

    Add-ADGroupMember `
        -Identity "Kojima-GG-GPO-Management" `
        -Members "Kojima-GG-Role-SysAdmin-DA"

    Add-ADGroupMember `
        -Identity "Kojima-GG-Group-Management" `
        -Members "Kojima-GG-Role-SysAdmin-DA"

    Add-ADGroupMember `
        -Identity "Kojima-GG-OU-Management" `
        -Members "Kojima-GG-Role-SysAdmin-DA"

    Write-Host "Membership applied successfully" -ForegroundColor Green
}
catch {
    Write-Host "Membership change failed." -ForegroundColor Red
    Write-Host $_.Exception.Message -ForegroundColor Yellow
}
