Task Groups


Task Groups and Delegations

Task Group Delegation Description
Kojima-GG-Task-User-Management Kojima-DL-Users-Manage User management
Kojima-GG-Task-Computer-Management Kojima-DL-Computers-Manage Computer object management
Kojima-GG-Task-GPO-Management Kojima-DL-GPOs-Manage Group Policy Object Management
Kojima-GG-Task-Group-Management Kojima-DL-Groups-Manage Group object management
Kojima-GG-Task-OU-Management Kojima-DL-OUs-Manage OU management


Creating Task Global Groups


Using Powershell to create Global Groups:


Import-Module ActiveDirectory

$GroupPath = "OU=Task Roles,OU=Kojima Roles,OU=Kojima,DC=kojima,DC=test"

try {
    New-ADGroup `
        -Name "Kojima-GG-Task-User-Managment" `
        -SamAccountName "Kojima-GG-Task-User-Managment" `
        -GroupCategory Security `
        -GroupScope Global `
        -Path $GroupPath `
        -Description "Task group for user management" `
        -ErrorAction Stop
...

    Write-Host "Global Task groups created successfully." -ForegroundColor Green
}
catch {
    Write-Host "Group creation failed." -ForegroundColor Red
    Write-Host $_.Exception.Message -ForegroundColor Yellow
}

Powershell ISE:


Validating via Powershell:


$DLGroups = @(
    "Kojima-DL-Users-Manage",
    "Kojima-DL-Computers-Manage",
    "Kojima-DL-GPOs-Manage",
    "Kojima-DL-Groups-Manage",
    "Kojima-DL-OUs-Manage"
)

foreach ($group in $DLGroups) {
    Write-Host "`nMembers of $group" -ForegroundColor Cyan
    Get-ADGroupMember -Identity $group |
    Select-Object Name, objectClass, SamAccountName |
    Format-Table -AutoSize
}




Delegating Permissions


User Management

ADUC -> Kojima Users -> Delegate Control -> Add Kojima-DL-Users-Manage


Computer Management

ADUC -> Kojima Computers -> Delegate Control -> Create a custom task to delegate

  • Read
  • Write
  • Read all properties
  • Write all properties
  • Create computer objects
  • Delete computer objects



GPO Management

GPMC -> kojima.test -> Group Policy Objects -> Delegation -> Kojima-DL-GPOs-Manage


GPMC -> kojima.test -> Delegation -> Add Kojima-DL-GPOs-Manage


Group Management

ADUC -> Kojima -> Delegate Control -> Add Kojima-DL-Groups-Manage -> Create a custom task to delegate

  • Create, delete, and manage groups
  • Modify the memberships of a group



OU Management


Powershell:


$OU = "OU=Kojima,DC=kojima,DC=test"
$Group = "KOJIMA\Kojima-DL-OUs-Manage"

dsacls $OU /G "$($Group):CCDC;organizationalUnit" /I:S
dsacls $OU /G "$($Group):WP;;" /I:S


Validation:


dsacls "OU=Kojima,DC=kojima,DC=test" | 
Where-Object { $_ -like "*OUs-Manage*" }

dsacls "OU=Kojima,DC=kojima,DC=test" | 
Where-Object { $_ -like "*OUs-Manage*" -or $_ -like "*organizationalUnit*" -or $_ -like "*CREATE*" -or $_ -like "*DELETE*" -or $_ -like "*WRITE*" } |
Select-Object -First 20
